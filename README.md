# Building

### Dependencies
- Premake5
- SDL System binary temporarily
- bgfx | bx | bimg

Currently tested and running only on Linux.

``` shell
git clone --recursive https://github.com/MunFaill/Umbra-Engine.git
cd Scripts && ./LinuxBuild.sh
```

# Contributing 

If you'd like to contribute in any way, feel free to create an Issue or a PR; all help and feedback are welcome.

## Code contribution

Clone this repository with:
``` shell
git clone --recursive https://github.com/MunFaill/Umbra-Engine.git
```

After making your changes, create a new branch:

``` shell
git add .
git commit -m "My amazing feature"
git push origin MyAmazingFeature
```
Finally, create a PR and you're done!

If you need help with the code structure, see this [code conduct.](CODE_CONDUCT.md)
