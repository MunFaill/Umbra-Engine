# Building

### Dependencies
- Premake5
- SDL System binary temporarily
- bgfx | bx | bimg

Currently tested and running only on Linux.

``` shell
git clone --recursive https://github.com/MunFaill/Umbra-Engine.git
cd Scripts && ./LinuxBuild.sh
```
